# 密码输入提示

## ⚠️ 重要提示

当程序要求输入**两步验证密码**时：

1. **密码输入是隐藏的**（不会显示字符）
   - 这是正常的安全行为
   - 直接输入密码，即使看不到字符也没关系
   - 输入完成后按回车

2. **如果确实无法输入**：
   - 确保光标在终端窗口中
   - 尝试点击终端窗口使其获得焦点
   - 直接输入密码（即使看不到），然后按回车

## 🔄 如果登录失败

### 方法 1：重新运行
```bash
python3 auto_fetch.py
```

### 方法 2：删除会话文件重新登录
```bash
# 删除旧的会话文件
rm gary_search_session.session

# 重新运行
python3 auto_fetch.py
```

### 方法 3：如果不需要两步验证密码
如果你的账号没有设置两步验证，程序可能不会要求输入密码。

## 💡 常见问题

**Q: 输入密码时看不到字符？**
A: 这是正常的！安全机制会隐藏密码。直接输入后按回车即可。

**Q: 输入后没有反应？**
A: 确保按了回车键。如果还是不行，尝试重新运行程序。

**Q: 验证码收不到？**
A: 检查手机号格式是否正确（需要国家代码，如 +86）。
